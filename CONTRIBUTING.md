# Contributing to VestRoll Mobile

We welcome contributions to the VestRoll Mobile Application!

## Code of Conduct
By participating, you agree to follow our [Code of Conduct](./CODE_OF_CONDUCT.md).

## How to Contribute
1. Fork the repo and create your feature branch.
2. Follow [Effective Dart](https://dart.dev/guides/language/effective-dart) guidelines.
3. Ensure all tests pass (`flutter test`).
4. Format your code (`dart format .`).
5. Open a Pull Request with a clear description of your changes.

## Development Setup
Refer to the [Quick Start](../README.md#quick-start) for environment setup instructions.
